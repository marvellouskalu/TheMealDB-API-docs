# Error Codes

The MealDB API uses standard REST Status codes


| Code                  | Description                                          
| :----------------     | :------:                                             
| 1XX                   |   Informational repsonse                
| 2XX                   |   Successful                                      
| 3XX                   |  Redirection                                   
| 4XX                   |  Client error                                          
| 5XX                   |   Server error
